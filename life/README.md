# Life Records

This directory is where local ontology records live.

The public skeleton keeps these folders empty. Copy templates from `templates/`
when creating private records.

Recommended conventions:

- Use stable snake_case IDs.
- Keep raw source files under `raw_data/`.
- Use `*_ref` and `*_refs` for object links.
- Add `confidence` and `needs_user_review` for uncertain interpretations.
- Keep current state separate from append-only status updates.
