# Project Instructions

This repository can be used by AI coding agents that read repository
instructions.

Read the shared operating model first:

@.codex/AGENTS.md

Canonical files:

- `.codex/life_ontology.yaml`: object, link, action, and query type definitions.
- `.codex/import_rules.yaml`: local import and raw data handling rules.
- `templates/`: public-safe templates for local records.

Repository rules:

- Keep raw personal records out of git.
- Keep the ontology lightweight and local-first.
- Add validation before adding automation.
