# Project Instructions

This repository is a public-safe skeleton for a local life ontology agent.

When working in this repository:

- Read `.codex/AGENTS.md` first.
- Treat `.codex/life_ontology.yaml` as the canonical ontology draft.
- Keep the setup lightweight. Do not scaffold an app, database, or service unless asked.
- Keep personal records local. Do not send or commit private life records unless explicitly requested.
- Prefer small, reviewable edits and meaningful git checkpoints.
